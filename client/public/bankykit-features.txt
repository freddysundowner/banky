MEMBER MANAGEMENT
-----------------
- Member registration with full KYC (ID, passport, photos)
- Unique member/account number generation
- Member profiles with next-of-kin and beneficiary details
- Member grouping (groups, Chamas, clubs)
- Member status management (active, suspended, exited)
- Duplicate ID number detection
- Member statements and transaction history
- Member search and filtering
- Mobile self-registration via member app
- OTP-based member identity verification


LOAN MANAGEMENT
---------------
- Configurable loan products (personal, business, emergency, group, etc.)
- Loan application, review, and approval workflow
- Multi-level approval (maker-checker)
- Loan disbursement to M-Pesa or bank account
- Reducing balance and flat-rate interest computation
- Flexible repayment schedules (daily, weekly, monthly)
- Loan instalment tracking
- Loan restructuring and rescheduling
- Loan top-ups
- Guarantor management
- Collateral recording and valuation
- Loan defaulting and write-off processing
- Overdue loan tracking and notifications
- Portfolio at Risk (PAR) reporting
- Loan aging analysis
- Auto loan repayment deduction from savings

- Local Postgres provisioning (no dependency on cloud DB services)
- Neon serverless Postgres provisioning for SaaS
- Multi-tenant data isolation
- Python FastAPI backend (high performance)
- React / TypeScript frontend (responsive web app)
- Flutter mobile app (Android & iOS)
- Separate admin panel
- Public landing page with CMS
- Role-based API security
- Session-based authentication with secure cookies
- Audit middleware on all write operations
- Background job scheduler (trial checks, maturity alerts, reminders)
